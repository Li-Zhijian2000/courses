2. `(x & 7) != 7 || (x << 29 < 0)`  $\checkmark$

   ```c
   	(x & 7 != 7) || (x << 29 < 0)
   => 	!((x & 7 == 7) && (x << 29 >= 0))
   ```

   当 `x & 7 == 7` 为假时，`(x & 7 == 7) && (x << 29 >= 0)` 为假，原式为真

   当 `x & 7 == 7` 为真时，有 `x == 7` ，此时 `x << 29 = 0xe0000000 < 0`  ，故 `(x & 7 == 7) && (x << 29 >= 0)` 为假，原式为真

   终上，原式为永真式

4. `x < 0 || -x <= 0`  $\checkmark$ 

    如果 $x < 0$ 不成立，则 $x >= 0$ ，即 $x \in [0, 2^{31} - 1]$ 

    所以 $-x \in [-(2^{31} - 1), 0] \sub [INT\_MIN, 0]$ 

5. `x > 0 || -x >= 0`  $\times$ 

    example: 

    ```c
    x = 0x80000000
    ```

6. `x + y == uy + ux`   $\checkmark$  

    有符号加法和无符号加法的位级表示相同

7. ` x * ~y + ux * uy == -x ` $\checkmark$ 

    ```c
    	x * ~y + ux * uy == -x
    =>	ux * uy == -(~y + 1) * x
    =>	ux * uy == x * y
    ```

    有符号乘法和无符号乘法的位级表示相同

``` c
float fpwr2(int x) {
    unsigned exp, frac;
    unsigned u;
    
    if (x <= -150) {
        exp = 0;
        frac = 0;
    }
    else if (x <= -127) {
        exp = 0;
        frac = 1 << (x + 150 - 1);
    }
    else if (x <= 127) {
        exp = x + 127;
        frac = 0;
    }
    else {
        exp = 255;
        frac = 0;
    }
    u = exp << 23 | frac;
    return u2f(u);
}
